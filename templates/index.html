ROUND 3 - TREASURE HUNT

Project Rewind: Monaco GP Memory Collapse

🚨 CHRONO EMERGENCY BRIEFING:
A brilliant scientist created a machine called Project Rewind to preserve school memories for a friend suffering from early-stage dementia. But during a test on May 26th, 2024, the machine became unstable. The memories began overlapping and collapsing, merging with a live broadcast of the Monaco Grand Prix. One of your teammates is trapped inside the collapsing memory track!

As the system crashes, your friend is slowly fading from existence. You must enter Project Rewind, solve riddles at various turns of the street circuit, collect lost memory fragments to unlock Security Clearance level 3, and pull your friend out before the memory timeline completely collapses!
